Que aprenderas?

  • Fundamentos, sintaxis y buenas prácticas de CSS.
  • Selectores, pseudoclases, modelo de caja.
  • Manejo del texto y del color.

Objetivo

Dar las herramientas necesarias tanto teoricas como practicas para adentrarse en el mundo de la tecnologia webdemostrando al final de este cursos lo aprendido con el proyecto que desarollaremos. Este curso se lleva de manera simultanea con el curso HTML desde cero

Requisitos

No requieres tenener ningun conocimiento ya que este curso la base para los siguientes que son mas avanzados.Este curso se lleva de manera simultanea con el curso HTML desde cero

A quien va dirigido?

A todo a aquel que desee adentrarse en el mundo de la tecnologia web, pueden venir del area de diseñocomo de la parte de desarrollo ó no saber absolutamente nada ya que el curso es desde cero.
