Sass - Preprocesados CSS

Sass le da la posibilidad a CSS de tener las ventajas de un lenguaje de Programación maximizando su uso.

JavaScript Avanzado

Git - Control de Versiones

Git es un sistema de control de versiones que nos va a servir para trabajar en equipo de una manera simple.
