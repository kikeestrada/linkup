Descripción

En este curso crearemos una app llamada postme la cual consistirá en el envío de mensajes, notas ycomentarios; para ello usaremos como framework de diseño Material Design Lite, la plataforma de Firebase y su base de datos Firestore, PouchDB, Express y Git desde cero usando html, css y javascript básico. ¿Quienes pueden tomar este curso? Aquellas personas que tienen un su sitio o página web y quieran modernizar implementando las últimas tecnologías creadas para las aplicaciones web.

Detalles

  • Duraccion: 1 Mes.
  • Horas: 16 - una clase semanal de 4 horas
  • Workshop - Proyecto Final.
  • Costo: $ 000.000 ivi.
  • Consulte nuestras facilidades de pago.

Que aprenderas?

  • Aprenderás a crear apps que no dependan de la conexión a internet para funcionar, para esto usaremos estrategias de red.
  • Ganarás presencia nativa instalando tu página o sitio web en tu dispositivo móvil utilizando el web manifest y pwacompact.
  • Podrás hacer uso de las notificaciones push cuando tu aplicación este activa o cerrada.

Objetivo

Capacitar al estudiante en una tecnologia de punta que permitirá emular en nuestro navegador nuestraaplicación web como una aplicación nativa

Requisitos

Para este curso es escencial tener los conocimientos adquiridos en los cursos:

  • Desarrollo web Introductorio
  • Desarrollo web Intermedio
  • Desarrollo web Avanzado

A quien va dirigido?

A todo a aquel que desee adentrarse y actualizarse en el mundo de la tecnologia web.
