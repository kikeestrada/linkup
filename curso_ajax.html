Curso Nombre

Description

Ajax es una tecnologia que nos permite consumir e imprimir información sin recargar la página. Veremos susimplementaciones modernas con .fetch() y promises. También aprenderás WebSockets, una tecnología que mantiene un canal de comunicación en tiempo real entre el cliente y el servidor.Duracion 12 horas

Que aprenderas?

  • Consumir APIs con Ajax
  • Utilizar fetch() y promises
  • Dibujar en el DOM usando datos JSON
  • Comunicación en tiempo real con el servidor

Que necesitas saber?

  • HTML
  • CSS
  • JS
  • JS en el Navegador
